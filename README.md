# TaleName Extension

Welcome to the official extension for the first unofficial username database of Hytale!
Install it here :

Chrome Extension : COMING SOON

Firefox Extension : COMING SOON

# Build

run **npm run build** to get the Firefox and chrome build in "build" folder.

# Features

## 1. Username status fetching
   Get status of a username

## 2. Claim a username on your TaleName account
   Let you show to the rest of the world what usernames you have!

## 3. Interact with the community!
   Help us build a cool place for Hytale Users.
